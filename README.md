# Bardic Inspiration Tracker

A small Foundry VTT module for D&D 5e that attaches a bardic inspiration tracker panel to bards character sheets.

## Features

- Clicking a party member grants them bardic inspiration.
- Each tracked party member shows inspiration status.
- Party members clicking the icon consume bardic inspiration and rolls the bard's die.

## Notes

- Built for Foundry VTT v13 and D&D 5e worlds.
- The module stores inspiration state on each Actor via flags.
